Gesture Object

A gesture object is passed to the action mapper whenever JSHG sucessfully detects hand gesture. This object contains following attributes and methods.

# Name Type Description
1 isLeft boolean
2 isRight boolean
3 isUp boolean
4 isDown boolean
5 nFingers number
6 handPos [number, number]
5 equals(gesture)
